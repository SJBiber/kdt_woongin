# 지하철 운행 모니터링 분석 리포트

**분석 일시**: 2026-01-07
**대상 구간**: 서울 지하철 (주요 분석: 3호선)
**데이터 범위**: 약 1.5시간 (7,300여 개의 레코드)

## 1. 배차 간격 정기성 분석 (Interval Regularity)
- **평균 배차 간격**: 약 11.8분
- **중앙값(Median)**: 7.9분
- **분석 결과**:
    - 대다수의 열차는 7~9분 간격으로 안정적으로 운행되고 있음.
    - 일부 10분 이상의 큰 간격은 데이터 수집 초기 단계의 공백으로 인한 이상치로 판단됨.

## 2. 지연 발생구간 탐지 (Delay Hotspots)
- **평균 체류 시간(Dwell Time)**: 약 82초
- **주요 정체 구간 (Hotspots)**:
    1. **오금역**: 평균 211초 (종착역 특성상 긴 체류 시간 발생)
    2. **구파발역**: 평균 109초 (회차 및 기점 운영 영향)
    3. **교대역, 고속터미널역**: 평균 90초 내외 (주요 환승역으로 인한 승하차 지연)
- **특이 사항**: 3258번 열차가 오금역에서 351초간 체류한 기록이 확인됨.

## 3. 회차 효율성 분석 (Turnaround Efficiency)
- **현재 상태**: 단기 데이터(1.5시간) 내에서 동일 열차 번호의 방향 전환(Direction Change) 패턴이 아직 충분히 식별되지 않음.
- **향후 계획**: 3시간 이상의 연속 데이터를 확보하여 종착역 도착 후 반대 방향 투입까지의 '순수 회차 시간' 산출 예정.

## 4. 급행/일반 열차 간섭 분석 (Congestion/Overtake)
- **대상 노선**: 1호선, 9호선 (급행 운행 노선)
- **분석 데이터**: 약 580건의 급행 열차 로깅 확인.
- **향후 계획**: 동일 선로상에서 급행열차가 일반열차에 의해 서행하는 '추월 대기' 패턴 탐지 알고리즘 적용 예정.
