## 📈 데이터 활용 방법

### 1. 전체 데이터 조회
```bash
cd scripts
python3 view_data.py
```

### 2. 특정 날짜 시계열 추이
```bash
python3 check_timeline.py
```

### 3. SQL 직접 조회
```sql
-- 최신 트렌드
SELECT upload_date, total_views, views_growth_rate
FROM daily_video_trends
WHERE keyword = '임성근 쉐프' 
  AND collected_date = CURRENT_DATE
ORDER BY upload_date DESC;

-- 특정 날짜 추이
SELECT collected_date, total_views, views_growth
FROM daily_video_trends
WHERE keyword = '임성근 쉐프' 
  AND upload_date = '2026-01-19'
ORDER BY collected_date;
```
