## 💡 핵심 인사이트 도출 방향

### 1. 논란의 성격 비교
```
백종원: 사업적 문제 (가맹점, 식품위생)
       → 시스템 개선으로 회복 가능
       → 개인 인격 공격은 상대적으로 적음

임성근: 개인 범죄 행위 (폭행)
       → 인격적 신뢰 붕괴
       → 회복이 더 어려울 가능성
```

### 2. 회복 속도 예측
```
백종원: 10개월 만에 부정 85% → 45% (40%p 감소)
       → 월평균 4%p 감소

임성근 예측:
- 낙관적: 백종원과 유사 (10개월 후 38%)
- 현실적: 더 느림 (12개월 후 45%)
- 비관적: 훨씬 느림 (15개월 후 50%)
```

### 3. 대응 전략 제안
```
백종원 사례에서 배운 점:
✓ 빠른 공식 사과
✓ 구체적인 개선 계획 발표
✓ 피해자 보상
✓ 일정 기간 자숙

임성근 권장 전략:
1. 진정성 있는 사과 (형식적 X)
2. 법적 책임 수용
3. 방송 활동 장기 중단
4. 사회 공헌 활동
```
