# PPT 초안 제작 계획

## 디자인 방향
- **톤**: 데이터 분석 보고서 / 심각한 여론 분석 주제에 맞는 차분하고 전문적인 디자인
- **색상 팔레트**: 다크 네이비(#1C2833) + 레드(#E33737) + 그린(#00CC96) + 블루(#636EFA) + 화이트(#F4F6F6)
- **폰트**: Arial (웹 안전 폰트)
- **레이아웃**: 16:9

## 슬라이드 구성 (총 10장)

### 1. 표지
- 프로젝트 제목: "임성근 음주운전 논란 여론 분석"
- 부제: "백종원 사례 비교를 통한 데이터 기반 분석"
- 발표자, 날짜

### 2. 프로젝트 개요
- 분석 배경: 임성근 음주운전 고백 (2026.01.19)
- 분석 목적: 논란 심각성 평가, 백종원 사례와 비교, 여론 변화 예측(시간이 지날수록 관심도가 떨어질것)
- 분석 데이터: YouTube 댓글 (임성근 ~31,500개, 백종원 ~13,900개)

### 3. 분석 방법론
- 3단계 파이프라인: 수집 → 정제 (Kiwi 형태소 분석) → LLM 분석 (DeepSeek-V3)
- 6가지 감정 분류: 지지/분노/중립/실망/조롱/질문
- 도구: YouTube API, Kiwi (kiwipiepy), DeepSeek LLM, Supabase, Streamlit, Plotly
- 비교 분석: 4가지 가설 검증 기반

### 4. 임성근 여론 핵심 지표
- 총 댓글 수, 부정 비율, 논란 후 부정 비율, 심각도 점수
- 긍정/부정/그외 도넛 차트

### 5. 논란 전후 감정 변화
- 6가지 감정의 논란 전/후 비율 비교 (Grouped Bar)
- 핵심 변화 포인트 텍스트

### 6. 백종원 비교 분석
- 두 사건의 감정 분포 비교
- 가설 검증: 분노 vs 조롱 비율
- 핵심 차이점 텍스트

### 7. 시계열 추이 분석
- 임성근: 논란 전후 1주일 부정 비율 변화 (Line Chart)
- 백종원: 월별 부정 비율 장기 추이 (Bar Chart)

### 8. 관심지수 분석
- 영상 트렌드 데이터 (조회수, 좋아요, 댓글 수 추이)
- 전일 대비 증감률

### 9. 핵심 인사이트 & 결론
- 논란 성격 차이 (사업적 vs 개인 범죄)
- 여론의 과열 및 소강 주기 분석
- 대중 공감 패턴 및 부정 확산율

### 10. Q&A
- 감사 + 질의응답
