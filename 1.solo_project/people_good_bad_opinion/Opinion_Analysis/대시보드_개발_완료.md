# 백종원 vs 임성근 비교 대시보드 개발 완료

## 📊 대시보드 구성

### 멀티페이지 Streamlit 앱 구조

```
Opinion_Analysis/
├── dashboard_main.py          # 메인 홈 페이지
└── pages/
    ├── 1_전체_요약.py          # 임성근 여론 분석 전체 요약
    ├── 2_감성_비교분석.py      # 감성 비교분석 (핵심 페이지)
    ├── 3_추이_비교분석.py      # 감정 추이 비교분석
    ├── 4_상세_통계.py          # 감정별 상세 통계 및 데이터 탐색
    └── 5_영상_트렌드.py        # 관심지수 분석
```

## 🎯 주요 기능

### 1️⃣ 전체 요약 페이지
- 임성근 여론 분석 핵심 지표
- 감정 그룹 분포 (긍정/부정/그외)
- 논란 전후 비교 (1월 19일 기준)
- 6가지 감정 카테고리 상세 분포
- 감정별 워드클라우드 및 키워드 Top 10

### 2️⃣ 감성 비교분석 페이지 ⭐
- **핵심 지표 비교 테이블**
  - 임성근 (현재) vs 백종원 (논란시) vs 백종원 (현재)
  - 부정 비율, 긍정 비율, 심각도 점수

- **감정 분포 비교 차트**
  - 3개 시점 나란히 비교

- **4가지 가설 검증** 📈
  - 가설 1: 논란 시 관심도 2배 증가
  - 가설 2: 분노·실망 댓글 주도
  - 가설 3: 1주 후 조롱 증가
  - 가설 4: 이슈 발생 후 관심도 감소

- **주요 이슈 비교**
  - 키워드 분석 기반 이슈 파악

- **가설 종합 요약 테이블**
  - 논란의 성격 비교
  - 회복 속도 예측
  - 권장 대응 전략

- **주단위 여론 변화 추이 분석**

### 3️⃣ 추이 비교분석 페이지
- 날짜별 감정 변화 라인 차트
- 일별 댓글 수 추이 (관심도)
- 주간 트렌드 분석
- 논란 시점 표시 (임성근: 1/19)
- **감정별 상세 분석**
  - 6가지 감정 비율
  - 감정별 영향력
  - 논란 전후 감정 변화
- **감정별 고정 색상 맵 적용**

### 4️⃣ 상세 통계 페이지
- 감정별 상세 통계 테이블
- 영향력 분석 (좋아요 Top 10 댓글)
- 감정별 좋아요 분포 (박스/바이올린 플롯)
- 가중 평균 감성 점수
- 댓글 길이 분석
- 키워드 공출현 분석
- 데이터 필터링 및 CSV 다운로드

### 5️⃣ 영상 트렌드 페이지
- **KPI 카드**
  - 영상 수 / 조회수 / 좋아요 / 댓글
  - 전일 대비 delta 표시
- **수집일별 종합 테이블**
- **통계 추이 차트**
  - 조회수 / 좋아요 / 댓글 탭별 구성
- **증감률 비교 차트**

## 📊 여론 변화 예측 알고리즘

### 백종원 실제 주간 데이터 기반
- 백종원 실제 주간(weekly) 여론 데이터를 활용하여 분석
- 논란 시기 (2025년 2-3월): 부정 비율 약 85%
- 현재 (2026년 1월): 부정 비율 약 45%
- 경과 기간: 약 10개월
- 월평균 감소율: 약 4%p

### 여론 추이 감지
- "개선" 트렌드 감지: 부정 비율이 감소하는 추세
- "악화" 트렌드 감지: 부정 비율이 증가하는 추세
- 양방향 변화를 모두 감지하여 종합적인 여론 흐름 파악

### 임성근 예측 모델
```python
# 백종원 실제 주간 데이터 기반 패턴 적용
months = [0, 1, 3, 6, 10]
recovery_rate = [1.0, 0.94, 0.82, 0.65, 0.53]

임성근_예측 = 현재_부정_비율 * recovery_rate
```

### 50% 이하 도달 시점
- 선형 보간법으로 계산
- 백종원 패턴 적용 시 약 8-10개월 예상
