당신은 데이터 엔지니어링 및 분석 마스터입니다.

다음과 같은 동작을 하는 프로그램을 개발합니다

**목표**
 - 네이버 검색 api를 사용해서 해당 키워드에 대한 일별 업로드 개수 파악
 - 전체 기간은 180일로 전부터 금일 - 1 (즉 전날) 까지 업로드한 블로그 개수를 추출
 - 되도록이면 api를 덜 사용하는 쪽은 원하는 기간의 전체 블로그 정보를 읽고 , 업로드 날짜를 가져와 그룹화 시키는 방향이 적합하다고 판단됨

**기술 스택**
 - 네이버 검색 api 사용
 - 파이썬 언어 사용
 - supabase 사용

**예상 동작 흐름도**
1. .env 파일 읽음 (네이버 검색 api 키 , supabase 키 )
2. 기간은 180일로 설정
3. 180기간동안의 블로그 정보를 api 호출하여 불러오고 정제
4. db에는 일별로 키워드에 대한 블로그 포스팅 수를 기록

**향후 계획**
1. 6개월 전 데이터를 작성했으니 , DAG를 사용하여 다음날부터는 오늘 포스팅된 블로그 수를 불러와 db에 저장하는 프로그램을 만들것임
