# ⏰ 내 컴퓨터 "스마트 알람" 설정하기 (누구나 1분 만에!)

안녕하세요! 매일 정해진 시간에 내 컴퓨터가 대신 일을 하게 만들고 싶으신가요? 
마치 우리가 아침에 알람 소리를 듣고 일어나듯이, 컴퓨터에게도 **"몇 시에 이 작업을 시작해!"**라고 알려주는 친절한 방법입니다.

---

## 💡 이게 뭔가요? (비유: 스마트 스케줄러)
매일 아침 루틴을 정해두는 것과 똑같습니다. 다만, 우리는 몸이 움직이지만, 컴퓨터는 **`example.py`라는 프로그램(자동화 로봇)**을 깨워서 일을 시킵니다.

- **방법**: 크론(Cron) 활용하기
- **장점**: 내 컴퓨터에서 바로 실행되므로 설정이 매우 직관적이고 빠릅니다.
- **포인트**: 설정만 해두면 매번 직접 실행할 필요가 없어집니다.

---

## 🚀 바로 따라 하기 (딱 3분!)

### 1단계: 컴퓨터에게 "스케줄판" 보여달라고 하기
컴퓨터의 터미널(검은 화면)을 열고 아래 명령어를 입력한 뒤 엔터를 치세요.
```bash
crontab -e
```
*(처음 실행 시 편집기를 선택하라고 나오면, `1`을 입력하고 엔터를 치시면 됩니다.)*

### 2단계: 작업 스케줄 입력하기
화면 맨 아래에 아래 내용을 복사해서 붙여넣으세요.
```bash
# 매일 오전 9시 정각에 내 프로그램을 실행합니다!
0 9 * * * /usr/bin/python3 /사용자/경로/example.py
```

**[시간 설정 읽는 법]**
- `0` : 0분
- `9` : 오전 9시
- `* * *` : 매일매일 반복!

### 3단계: 저장하고 완료하기
- 자판에서 `Esc` 키를 누르세요.
- 그 다음 `:wq` 라고 입력하고 엔터를 치세요. (저장하고 종료한다는 뜻입니다!)

---

## ✅ 정상 등록 확인
스케줄이 잘 등록됐는지 보려면 아래를 입력해보세요.
```bash
crontab -l
```
내가 방금 적은 내용이 보인다면 모든 준비가 끝났습니다! 🎉

---

## 💡 알아두면 좋은 점
1. **전원 확인**: 설정한 시간에 컴퓨터가 켜져 있어야 로봇이 출근해서 일을 시작할 수 있습니다.
2. **주소 확인**: "실행할 파일이 어디 있는지" 컴퓨터에게 정확한 주소를 알려주는 것이 중요합니다.
(예시: 0 9 * * * /usr/bin/python3 /배승재/집/example.py // 9시에 python3으로 배승재의 집에 있는 example.py를 실행해! 라는 느낌입니다~)
