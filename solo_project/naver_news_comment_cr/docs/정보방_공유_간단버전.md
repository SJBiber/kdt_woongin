## 📢 버전 2: 중간 (게시판용)

```
# 💡 클라우드 없이 데이터 수집 자동화하기 (무료)

데이터 수집 프로젝트 만들었는데 매번 수동 실행하기 귀찮으시죠?
AWS/GCP는 비용이 부담되고...

**무료로 자동화하는 2가지 방법**을 공유합니다!

## 방법 비교

| 항목 | Mac Cron | GitHub Actions |
|------|----------|----------------|
| 비용 | 무료 | 무료 |
| PC | 켜둬야 함 | 꺼도 됨 ✨ |
| 설정 | 1분 | 5분 |
| 로그 | 수동 | 자동 |
| 알림 | 없음 | 이메일 |

## ① Mac Cron (초간단)

Mac에 기본 내장된 스케줄러

설정 방법:
```bash
crontab -e
0 9 * * * /path/to/script.sh  # 매일 오전 9시
```

끝! 1분이면 완료

## ② GitHub Actions (강력 추천!)

GitHub에서 제공하는 무료 자동화 서비스

장점:
✅ PC 꺼도 됨 (GitHub 서버에서 실행)
✅ 월 2,000분 무료
✅ 로그 자동 저장
✅ 실패 시 이메일 알림

설정 방법:
1. GitHub Secrets에 API 키 등록
2. .github/workflows/collect.yml 파일 생성
3. Actions 탭에서 테스트

## 상세 가이드

📖 Mac Cron: AUTOMATION.md
📖 GitHub Actions (초심자): GITHUB_ACTIONS_TUTORIAL.md
📖 GitHub Actions (빠른 설정): GITHUB_ACTIONS.md

링크: https://github.com/SJBiber/kdt_woongin/tree/main/solo_project/youtube_trend_tracker/docs

## 추천

- Mac 항상 켜둠 → Cron
- 노트북 들고 다님 → GitHub Actions
- 팀 프로젝트 → GitHub Actions

더 이상 수동으로 실행하지 마세요! 😊
```

## 🎯 사용 가이드

### 어디에 어떤 버전을 쓸까?

| 플랫폼 | 추천 버전 | 이유 |
|--------|----------|------|
| 카카오톡 | 버전 1 | 짧고 간결 |

### 복사하는 방법

1. 원하는 버전의 코드 블록 복사
2. 플랫폼에 붙여넣기
3. 링크 부분을 실제 GitHub 링크로 교체

### 링크 교체

```
