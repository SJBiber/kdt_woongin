# 💡 클라우드 없이 자동화하는 방법 (무료)

안녕하세요! 데이터 수집 프로젝트를 자동화하고 싶은데 AWS나 GCP 같은 클라우드 서비스는 비용이 부담되시죠? 

**무료로 자동화할 수 있는 2가지 방법**을 공유합니다!

---

## 🎯 방법 비교

| 항목 | ① Mac Cron | ② GitHub Actions |
|------|-----------|-----------------|
| **비용** | 무료 (전기세만) | 완전 무료 |
| **PC** | 켜둬야 함 | 꺼도 됨 ✨ |
| **설정 시간** | 1분 | 5분 |
| **난이도** | 매우 쉬움 | 쉬움 |
| **로그 관리** | 수동 | 자동 |
| **실패 알림** | 없음 | 이메일 |
| **추천 대상** | Mac 사용자 | 모든 사람 |

---

## 📌 방법 ① Mac Cron (초간단)

> Mac에 기본 내장된 스케줄러. 설정 1분이면 끝!

### 장점
- ✅ 설치 불필요 (이미 있음)
- ✅ 설정이 정말 간단함
- ✅ 로컬에서 실행되어 디버깅 쉬움

### 단점
- ⚠️ Mac을 켜둬야 함
- ⚠️ 전기세 발생

### 설정 방법
```bash
# 1. Cron 편집기 열기
crontab -e

# 2. 아래 줄 추가 (매일 오전 9시 실행)
0 9 * * * /path/to/your/script.sh

# 3. 저장 후 종료 (vim: :wq, nano: Ctrl+X)
```

**끝!** 내일 오전 9시부터 자동 실행됩니다.

📖 **상세 가이드**: [AUTOMATION.md](https://github.com/SJBiber/kdt_woongin/blob/main/solo_project/youtube_trend_tracker/docs/AUTOMATION.md)

## 📌 방법 ② GitHub Actions (강력 추천!)

> GitHub에서 제공하는 무료 자동화 서비스. PC 없이도 작동!

### 장점
- ✅ **PC 꺼도 됨** (GitHub 서버에서 실행)
- ✅ 월 2,000분 무료 (충분함)
- ✅ 로그 자동 저장
- ✅ 실패 시 이메일 알림
- ✅ 언제 어디서나 실행 기록 확인

### 단점
- ⚠️ GitHub 계정 필요
- ⚠️ API 키를 GitHub Secrets에 등록해야 함

### 설정 방법 (5분)

#### 1단계: GitHub Secrets 등록
```
저장소 → Settings → Secrets and variables → Actions
→ New repository secret 클릭
```

필요한 Secret 4개:
- `YOUTUBE_API_KEY_1`: YouTube API 키 1
- `YOUTUBE_API_KEY_2`: YouTube API 키 2
- `SUPABASE_URL`: Supabase 프로젝트 URL
- `SUPABASE_KEY`: Supabase Service Role Key

#### 2단계: 워크플로우 파일 생성
`.github/workflows/collect.yml` 파일 생성 (예시 코드는 가이드 참조)

#### 3단계: 테스트
```
Actions 탭 → Run workflow 버튼 클릭
```

**끝!** 매일 자동으로 실행됩니다.

## 🤔 어떤 방법을 선택해야 할까?

### Mac Cron을 선택하세요 (이런 경우)
- Mac을 항상 켜두는 경우
- 로컬에서 바로 디버깅하고 싶은 경우
- 1분 안에 설정하고 싶은 경우

### GitHub Actions를 선택하세요 (이런 경우)
- PC를 자주 끄는 경우
- 노트북을 들고 다니는 경우
- 실행 기록을 웹에서 확인하고 싶은 경우
- 팀원과 공유하고 싶은 경우

## 💡 실전 팁

### 1. 두 가지 모두 사용하기
- **평일**: GitHub Actions (PC 꺼도 됨)
- **주말**: Mac Cron (로컬 테스트용)

### 2. 시간대 주의
- **Cron**: 한국 시간 그대로 사용
- **GitHub Actions**: UTC 기준 (한국 시간 -9시간)
  - 예: 오전 9시 실행 → `cron: '0 0 * * *'` (UTC 00:00)

### 3. 비용 걱정 NO!
- Cron: 전기세만 (미미함)
- GitHub Actions: 월 2,000분 무료 (일 5분 실행 시 400일치)
