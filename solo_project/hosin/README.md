# 호신용품 시장 및 범죄 데이터 분석 프로젝트

사회적 안전 이슈와 관련된 범죄 통계 및 호신용품 트렌드를 분석하는 프로젝트입니다.

## 📁 디렉토리 구조

- `scripts/`: 데이터 전처리 및 분석 스크립트
- `config/data/`: 원천 데이터 (엑셀, CSV 등)
- `docs/`: 분석 보고서 및 산출물 이미지
- `src/`: (분석용 공용 함수 예정)

## 🚀 시작하기

1. `scripts/` 내의 분석용 파이썬 스크립트 실행
2. `docs/` 내의 분석 보고서 확인

## 🛠️ 기술 스택
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
