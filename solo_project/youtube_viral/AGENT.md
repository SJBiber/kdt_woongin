# 🚀 Project: YouTube Viral Radar (YVR) - Advanced v2.5 (Snapshot Edition)
## API 할당량 최적화 및 ROI 중심 실시간 트렌드 분석 시스템

본 시스템은 YouTube API의 엄격한 할당량(Quota) 제한 환경에서 최대의 분석 효율을 내기 위해 설계되었습니다. 개별 영상을 장시간 추적하는 대신, 특정 시점의 검색 결과 셋(Set)을 대량 분석하여 트렌드의 **'파워'**와 **'신선도'**를 즉각적으로 도출하는 **Snapshot Analysis** 방법론을 채택합니다.

---

## 1. 프로젝트 목표 (v2.5)
*   **Snapshot-based Insight**: 단 한 번의 API 호출로 해당 키워드의 현재 성장력과 공급량을 즉시 판정.
*   **Quota Optimization**: 불필요한 추적(Tracking)을 제거하여 API 소모량을 기존 대비 80% 절감.
*   **High ROI Intelligence**: 할당량 1 unit당 도출되는 정보의 가치를 극대화하는 지표 설계.

## 2. 주요 분석 지표 (New Metrics)

### ① Keyword Topic Share (점유 지수)
*   현재 한국 인기 급상승 영상 300개 중 특정 키워드가 차지하는 비중.
*   **용도**: 해당 주제가 현재 유튜브 생태계에서 얼마나 주류(Mainstream) 트렌드인지 판정.

### ② Trend Power Score (성장 지수)
*   $$Power = \frac{\sum (ViewCount_i / HoursSinceUpload_i)}{N}$$
*   **용도**: 영상이 업로드된 후 시간당 평균 조회수 상승폭을 계산. 추적 없이도 현재의 가파른 상승세를 확인.

### ③ Upload Density (공급 밀도)
*   최근 검색 결과 50개 중 **24시간 이내** 업로드된 영상의 비율.
*   **용도**: 창작자들이 특정 주제에 대해 얼마나 공격적으로 영상을 공급하고 있는지(Surge) 판정.

## 4. 데이터 다양성 및 중립성 확보 전략 (Diversity-First)

특정 카테고리(예: 게임, 뉴스)가 유튜브 트렌드를 과점하더라도, 시스템은 **모든 분야**의 고른 데이터를 수집하기 위해 다음과 같은 전략적 구조를 가집니다.

### ① 카테고리별 계층 추출 (Stratified Extraction)
*   **작동 원리**: 전체 300개의 인기 영상을 카테고리(음악, 시사, 스포츠 등)별로 그룹화한 뒤, 각 그룹 내부에서 가장 빈도가 높은 키워드를 산출합니다.
*   **Round-Robin 선발**: 각 카테고리의 1순위 키워드를 하나씩 번갈아 가며 선발하여, 특정 카테고리가 상위권을 독식하더라도 리포트에는 다양한 분야의 이슈가 반드시 포함되도록 합니다.
*   **객관성 보장**: 인위적으로 키워드를 차단(Stopwords)하는 대신, 시스템 구조를 통해 데이터 스스로의 다양성을 확보함으로써 분석의 중립성을 유지합니다.

## 5. 데이터베이스 설계 철학 (Snapshot-First)
*   **비정규화**: 분석 속도를 위해 키워드 분석 결과를 단일 테이블에 집약.
*   **시계열 흔적**: 개별 영상의 변화량이 아닌, **키워드 전체의 파워 변화**를 추적.
*   **유연한 확장**: 새로운 필터링(예: 카테고리별 분석) 도입이 용이하도록 설계.
